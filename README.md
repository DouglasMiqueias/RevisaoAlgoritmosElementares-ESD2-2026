# Estrutura de Dados 2 — Atividades em Java

Este repositório reúne uma série de exercícios desenvolvidos em **Java** para a disciplina de **Estrutura de Dados 2**.

As atividades envolvem implementação e estudo de algoritmos clássicos de ordenação, recursão e manipulação de vetores, com foco em lógica de programação e análise de eficiência.

---

## Conteúdo das atividades

### Ex01 — Verificar ordem crescente
Implementação de uma função que verifica se um vetor `v[0..n-1]` está em **ordem crescente**.

### Ex02 — Inserção em vetor ordenado
Implementação de uma função que recebe um inteiro `x` e um vetor em ordem crescente, inserindo `x` na posição correta para **manter a ordenação**.

### Ex03 — Insertion Sort recursivo
Versão **recursiva** do algoritmo de ordenação por **inserção**.

### Ex04 — Selection Sort recursivo
Versão **recursiva** do algoritmo de ordenação por **seleção**.

### Ex05 — Ordenação decrescente
Função que reorganiza os elementos de um vetor inteiro em **ordem decrescente**, inspirada no algoritmo **Selection Sort**.

### Ex06 — Algoritmo mais rápido que inserção e seleção
Implementação de um algoritmo de ordenação mais eficiente que **Insertion Sort** e **Selection Sort**, com explicação sobre seu desempenho.

## Objetivo

O objetivo destas atividades é praticar conceitos fundamentais de **Estrutura de Dados**, especialmente:

- manipulação de vetores
- algoritmos de ordenação
- raciocínio recursivo
- comparação de eficiência entre algoritmos
